# Nutrition Diary 🍎

A mobile application for calorie counting developed with C# and .NET MAUI.

## Features

- 📊 Daily calorie tracking
- 🍽️ Meal diary with BZHU (proteins, fats, carbs)
- 👤 User registration and profiles
- 📖 Recipe collection
- 📈 Statistics and progress tracking

## Technology Stack

- .NET MAUI
- C#
- SQL Server
- XAML for UI

## Installation

1. Clone the repository
2. Open in Visual Studio 2022
3. Restore NuGet packages
4. Run on Windows Machine or emulator

## Database

The application uses SQL Server Express with the following main tables:
- Users
- Products  
- DiaryEntries
- Recipes

## Development

```bash
# Clone repository
git clone https://github.com/ВАШ_АККАУНТ/NutritionDiary.git

# Open in Visual Studio
start NutritionDiary.sln
```
